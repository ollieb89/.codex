Add three new exports to commands.js below the existing exports:

1. `MIN_SECRET_LENGTH = 8` -- minimum character length for generic fallback values to be considered secrets.

2. `SECRET_PATTERNS` -- an ordered array of pattern objects. Each object has:
   - `name` (string): identifier for the pattern
   - `regex` (RegExp with /g flag): the detection pattern
   - `alwaysRedact` (boolean): true for provider patterns (skip safe-value checks), false for generic fallback

   Array order (specific to generic):
   a. `anthropic`: `/\bsk-ant-[A-Za-z0-9_-]{20,}\b/g` -- MUST come before openai
   b. `openai`: `/\bsk-[A-Za-z0-9]{20,}\b/g`
   c. `github`: `/\bghp_[A-Za-z0-9]{36,}\b/g`
   d. `aws`: `/\bAKIA[A-Z0-9]{16}\b/g`
   e. `stripe`: `/\bsk_live_[A-Za-z0-9]{24,}\b/g`
   f. `pem`: `/-----BEGIN [A-Z ]+-----[\s\S]*?-----END [A-Z ]+-----/g`
   g. `jwt`: `/\beyJhbG[A-Za-z0-9_-]+\.eyJ[A-Za-z0-9_-]+\.[A-Za-z0-9_-]+/g`
   h. `bearer`: `/\bBearer\s+[A-Za-z0-9_\-.~+\/]+=*/g`
   i. `connection_string`: `/\b[a-z][a-z0-9+.-]*:\/\/[^\s:]+:[^\s@]+@[^\s]+/g`
   j. `generic_env`: `/\b([A-Z0-9_]*(API_KEY|TOKEN|SECRET|PASSWORD|CREDENTIAL|AUTH))\s*=\s*(\S+)/g` -- alwaysRedact: false

3. `isSafeValue(value)` -- returns true if value should NOT be redacted by generic fallback:
   - Numeric-only: `/^\d+$/.test(value)` returns true
   - File path: value starts with `/`, `./`, `../`, or `~`
   - Below minimum length: `value.length < MIN_SECRET_LENGTH`

Add SECRET_PATTERNS, isSafeValue, and MIN_SECRET_LENGTH to the module.exports object.
